This project explores a neuro-symbolic reward modeling framework that combines Differentiable Decision Trees (DDTs) with fuzzy logic constraints for learning interpretable reward functions from pairwise trajectory preferences. The results are very inconclusive, but the project investigates and raised some really interesting questions.
